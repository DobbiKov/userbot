# DobbiKov userbot for Telegram

## General information
Developer: Roman Zapotockiy[DobbiKov]\
Telegram Group: https://t.me/dobbikov_dev

## Stack
- Python
- PyRoGram

## SetUp
- Install Python
- pip install pyrogram
- create app on this [site](my.telegram.org) 
- create file `config.ini` and write:
```javascript
[pyrogram]
api_id = your_id
api_hash = your_hash
```
